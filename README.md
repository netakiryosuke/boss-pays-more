boss-pays-more
========================================

「上司（偉い人）が多めに払う」系の割り勘計算アプリです。（[Try it online](https://netakiryosuke.github.io/boss-pays-more/)）

支払いの重みづけに対応し、1円単位だけでなく1000円単位の割り勘もできます。

Requirements
----------------------------------------

- Node.js 24
- npm

Setup
----------------------------------------

```bash
git clone https://github.com/netakiryosuke/boss-pays-more.git
cd boss-pays-more
npm install
```

How to run (dev)
----------------------------------------

```bash
npm run dev
```

- http://localhost:3000/

How to build (static export)
----------------------------------------

```bash
npm run build
```

How to preview build locally
----------------------------------------

```bash
npm run build
npx serve out
```
